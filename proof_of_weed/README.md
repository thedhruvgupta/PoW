# Proof of Weed

**Proof of Weed** is a modern, decentralized application (dApp) that revolutionizes the way dispensaries manage their sales and payments. Built with Next.js, Tailwind CSS, and ethers.js, this platform allows users to connect their cryptocurrency wallets, browse products from various dispensaries, and make payments using both traditional methods (via Stripe) and cryptocurrencies.

## âœ¨ Features

- **Wallet Integration**: Seamlessly connect to Ethereum wallets like MetaMask to manage funds and make payments.
- **Dispensary and Product Browsing**: Explore a curated list of dispensaries and their product offerings in a clean, user-friendly interface.
- **Dual Payment System**: Choose between paying with a credit card (powered by Stripe) or with cryptocurrency for a flexible checkout experience.
- **Modern UI**: A sleek, responsive, and intuitive user interface built with Tailwind CSS, featuring a dark mode for comfortable browsing.
- **Decentralized Payments**: Cryptocurrency payments are sent directly to the dispensary's wallet, ensuring a secure and transparent transaction.

## ðŸ“¦ Setup and Installation

To get started with the Proof of Weed dApp, follow these simple steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/proof-of-weed.git
   cd proof-of-weed
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up environment variables**:
   Create a `.env.local` file in the root of your project and add the following variables:
   ```env
   NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=your_stripe_publishable_key
   ```
   Replace `your_stripe_publishable_key` with your actual Stripe publishable key.

4. **Run the development server**:
   ```bash
   npm run dev
   ```

   Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
